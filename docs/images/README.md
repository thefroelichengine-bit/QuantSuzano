# Visualization Assets

This folder contains all the visualization images used in the main README.

## Images (12 plots)

These images are generated by running:
```bash
python run_pipeline_safe.py all-robust
```

Original source: `data/out/plots/*.png` (copied here for Git/GitHub visibility)

### Available Plots:

1. **backtest_pnl.png** - Backtest PnL, positions, and drawdown
2. **correlation_heatmap.png** - Correlation matrix heatmap
3. **distributions.png** - Return distributions
4. **levels.png** - Time series levels (all variables)
5. **pred_vs_actual_splits.png** - Model predictions vs actual (train/val/test)
6. **residual_diagnostics.png** - Comprehensive residual analysis
7. **returns.png** - Log returns time series
8. **rolling_corr_suzb_r_pulp_brl_r.png** - Rolling correlation
9. **scatter_actual_vs_pred.png** - Scatter plot of predictions
10. **signals.png** - Trading signals on price chart
11. **synthetic_vs_actual.png** - Synthetic index tracking
12. **zscore_analysis.png** - Z-score trading signals

## Regenerating Images

To regenerate these images:

```bash
# Run the complete analysis pipeline
python run_pipeline_safe.py all-robust

# Copy new plots to docs
xcopy "data\out\plots\*.png" "docs\images\" /Y
```
